{{experienceHeader.projDesc}} Freight Rate Exchange Database (FRED) is web-based audit platform. It is used for tracking, pre-calculating, auditing shipments as well as releasing them for payment. It can be used with in or across companies. Using FRED, shipment data files are received via an interface and that data is matched with an electronic billing file from the carrier. Based on the matching results discrepancies are solved if any or processed further to close the audit loop. The carrier also has the option to declare any extra or unforeseen costs that he may have via extra costs tool. FRED provides solutions which are always set up to the specific needs of the customer.
